Comandos por Módulo

  1. show ip interface brief
  2. configure terminal
  3. enable secret <clave>
  4. ip route 0.0.0.0 0.0.0.0 <gateway>

Definiciones Técnicas

NAT
Traducción de direcciones de red para permitir acceso externo.
ACL
Listas de control de acceso que filtran tráfico según reglas.
DHCP
Protocolo que asigna direcciones IP automáticamente.
VTP
Protocolo para administrar VLANs en redes Cisco.

Interfaces del Router

  • GigabitEthernet
    1. interface GigabitEthernet0/0
    2. ip address 192.168.1.1 255.255.255.0
  • Serial
    1. interface Serial0/0/0
    2. clock rate 64000
